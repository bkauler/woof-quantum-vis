Limine Installer

Limine is a very simple and easy-to-use bootloader or boot-manager, for both legacy-BIOS and UEFI firmware computers.

A Limine installation requires only two or three files. For legacy-BIOS booting, 'limine.sys' and 'limine.cfg'. For UEFI booting, 'BOOTX64.EFI' and 'limine.cfg'.

Project page:

https://github.com/limine-bootloader/limine/tree/v3.0-branch

Limine Installer is a GUI to install Limine.
