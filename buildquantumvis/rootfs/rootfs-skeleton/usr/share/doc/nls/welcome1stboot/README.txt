welcome1stboot is the welcome window on first boot.

It is a binary executable, written in BaCon. The executable is at /usr/sbin
and the source code is /usr/sbin/welcome1stboot.bac and compile instructions
/usr/sbin/welcome1stboot.pupdev. The latter two are in the 'devx' SFS.

/usr/share/doc/nls/welcome1stboot/welcome1stboot.pot is also in the 'devx'.

MoManager, the Translation manager, reads the .pot file.
