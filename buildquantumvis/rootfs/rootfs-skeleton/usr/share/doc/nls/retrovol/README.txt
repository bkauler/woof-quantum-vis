retrovol is a binary executable, a sound mixer and tray volume control, written by Puppy Forum member pizzasgood.

The .pot was obtained from this 0.12 source:
http://www.browserloadofcoolness.com/software/puppy/PETget/retrovol-0.12.tar.gz

Get latest source:
http://puppylinux.svn.sourceforge.net/viewvc/puppylinux/retrovol/

MoManager can be used to create a translation.
