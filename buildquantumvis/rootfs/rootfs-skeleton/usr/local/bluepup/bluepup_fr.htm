BluePup Gestionnaire Bluetooth

BluePup est un emballage graphique autour de l'utilitaire en ligne de commande «bluetoothctl», qui fait partie du paquet «bluez».

Fenêtre de l'Interface Graphique

Dans la zone «Commande», toute commande acceptée par bluetoothctl peut être saisie et le résultat sera affiché dans le «Journal de sortie».

Mais, vous n'aurez probablement pas besoin de taper de commandes, car cliquer sur les boutons «RECHERCHER» et «ENREGISTRER» sont des opérations simples qui fonctionneront dans la majorité des cas.

Si de nouveaux appareils sont détectés, ils seront affichés dans «Appareils trouvés», c'est une liste à partir de laquelle l'utilisateur peut faire un choix.

Un appareil qui a été associé avec succès ira dans la liste «Appareils enregistrés». Il devra toujours être connecté pour être utilisé. Une coche orange indique associé, une coche verte indique connecté.

Bien sur, un appareil qui a été connecté une fois, le sera automatiquement à l'avenir, s'il est allumé avant le démarrage ou après.

Agent

L'agent est un démon utilisé pour négocier avec l'appareil distant pendant l'association. Cela peut nécessiter la saisie ou la confirmation d'un code PIN sur l'appareil distant.
BluePup affichera le code PIN requis dans certains cas, comme pour un clavier.

Appliquette de BluePup

Indique actuellement uniquement si le démon Bluetooth «bluetoothd» est en cours d'exécution ou non. De cliquer sur l'icône lance l'interface graphique de BluePup.
